# kickstart-api
Start pre-defined scripts and programs on your home infrastructure via CLI.

Get started with
```shell
# Windows
set FLASK_APP=app.py
flask run

# Linux
export FLASK_APP=app.py
flask run
```

Setup the CLI with
```shell
# Windows
virtualenv venv
.\venv\Scripts\activate
pip install --editable .

# Linux
virtualenv venv
.\venv\bin\activate
pip install --editable .
```
